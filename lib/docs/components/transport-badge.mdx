## API Reference

| Prop              | Type                                             | Default | Description                                           |
| ----------------- | ------------------------------------------------ | ------- | ----------------------------------------------------- |
| `system`          | `"SG"` (required)                                | -       | **Required.** Transport system identifier.            |
| `stationCode`     | `string \| string[]` (required)                  | -       | **Required.** Station code(s) for the badge.          |
| `stationName`     | `string`                                         | -       | Optional station name to display.                     |
| `showStationName` | `boolean` (default: `false`)                     | `false` | Whether to show the station name alongside the badge. |
| `size`            | `"xs" \| "sm" \| "md" \| "lg"` (default: `"md"`) | `"md"`  | Size variant of the badge.                            |
| `className`       | `string`                                         | -       | Additional CSS classes for styling.                   |
